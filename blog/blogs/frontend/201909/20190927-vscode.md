## 常见问题

### 路径别名设置

在 vscod 中一般是可以通过`ctrl+左击`或`alt+左击`操作来进行引用模块跳转，但是如果是通过`webpack`设置的别名组成的路径是无法跳转，我们需要进行设置。

1. 在项目根位置创建`jsconfig.json`。
2. 配置`jsconfig.json`:
```json
{
  "include": ["./src/**/*"],
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["src/*"]
    }
  },
  "exclude": ["node_modules", "build"]
}
```
3. 重启 vscode。
